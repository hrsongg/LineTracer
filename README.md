### Competiton Rule
1. 만들어진 Track에서 2바퀴 연속으로 Line Tracing 후 Lap time 측정하여 순위 결정
2. Single Sensor로 Line Tracing할 경우 advantage(-10s) 제공
- 중간에 멈추거나 line을 이탈하는 경우 실격으로 간주
- 주행 중 리모컨 추가 조정 불가능
